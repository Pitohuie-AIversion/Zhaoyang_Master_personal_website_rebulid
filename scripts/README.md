# Scripts Directory

This directory contains various utility scripts for maintenance, testing, and deployment.

## Structure

- **`cleanup_and_checks/`**: Scripts for checking database structure, translations, and general health checks.
- **`debugging/`**: Scripts specifically used for debugging JSON parsing or specific errors.
- **`deployment/`**: Scripts related to Vercel or other deployment tasks.
- **`security/`**: Scripts for security scanning, encryption testing, and key management.
- **`i18n/`**: Scripts for internationalization checks and alignment.

## Usage

Most scripts can be run with Node.js:
```bash
node scripts/cleanup_and_checks/complete-self-check.js
```
